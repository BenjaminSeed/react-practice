# React Practice Project

## ABOUT

A small project designed to use some of React's main features such as:

- Components
- Props
- State (useState hook)
- Passing Functions

The project also uses css styling.

### Technologies and Testing used

- [React](https://reactjs.org/)

### How to run the app

- `Clone` down this Repo
- Run `npm install`
- Run `npm start`
